# Quick Start Guide

Get up and running with AGI Workforce in under 5 minutes!

## 3-Step Installation

### Step 1: Download (1 minute)

Visit **[agiworkforce.com/download](https://agiworkforce.com/download)** and click your platform:

- 🍎 **macOS** - Download .dmg file
- 🪟 **Windows** - Download .exe installer
- 🐧 **Linux** - Download .AppImage or .deb

### Step 2: Install (1 minute)

**macOS:**

1. Open the .dmg file
2. Drag AGI Workforce to Applications
3. Right-click → Open (first time only)

**Windows:**

1. Run the .exe installer
2. Follow the installation wizard
3. Launch from Start Menu

**Linux:**

```bash
# AppImage
chmod +x AGI-Workforce.AppImage
./AGI-Workforce.AppImage

# Debian/Ubuntu
sudo dpkg -i agi-workforce.deb
```

### Step 3: Set Up Account (2 minutes)

1. Launch AGI Workforce
2. Click **"Create Account"**
3. Enter your email and password
4. Verify email (check your inbox)
5. Choose **Free Plan** (no credit card needed)

Done! You're ready to start.

## Your First 3 Tasks (5 minutes)

### Task 1: Simple Question (1 minute)

In the chat, type:

```
What can you help me with?
```

Press **Enter** and watch AI respond!

**What you learned:** Basic chat interface

---

### Task 2: File Operation (2 minutes)

Type this:

```
List all files in my Downloads folder organized by type
```

1. AI will ask permission to access Downloads
2. Click **"Allow"**
3. See your files organized automatically!

**What you learned:** File operations and permissions

---

### Task 3: Generate Code (2 minutes)

Type this:

```
Write a Python function to calculate factorial with example usage
```

You'll get:

- Complete, working code
- Comments explaining how it works
- Example showing how to use it

Click the **copy button** to save the code!

**What you learned:** Code generation

## 5 Essential Keyboard Shortcuts

Learn these and you'll be 10x faster:

| Shortcut           | What it does                              |
| ------------------ | ----------------------------------------- |
| `Cmd/Ctrl + K`     | Open command palette (access everything!) |
| `Cmd/Ctrl + Enter` | Send your message                         |
| `Alt + P`          | Switch AI models                          |
| `Cmd/Ctrl + /`     | Show all keyboard shortcuts               |
| `Escape`           | Cancel or close anything                  |

**Pro Tip:** Press `Cmd/Ctrl + K` and type what you want - it's the fastest way to do anything!

## Common Questions

**Q: Do I need API keys?**
A: No! Just use your subscription - all models included.

**Q: Can I use it offline?**
A: Partially. Install Ollama for completely offline AI.

**Q: Is my data safe?**
A: Yes! Everything stored locally by default. You control access.

**Q: How many messages can I send?**
A: Free plan: 10/day. Paid plans: 100 to unlimited.

**Q: Can AI access my files?**
A: Only folders you explicitly allow in Settings → Privacy.
