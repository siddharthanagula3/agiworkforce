## Learning Paths

### Path 1: Complete Beginner (1 week)

**Day 1: Basics**

- Read [Quick Start Guide](QUICK_START.md)
- Complete [Getting Started](GETTING_STARTED.md)
- Try your first 3 tasks

**Day 2: Chat Mastery**

- Learn [Chat Interface](USER_GUIDE.md#chat-interface)
- Try different [AI Models](USER_GUIDE.md#model-selection)
- Practice [Keyboard Shortcuts](KEYBOARD_SHORTCUTS.md)

**Day 3: File Operations**

- Read [File Management](USER_GUIDE.md#file-management)
- Watch [File Operations Tutorial](VIDEO_TUTORIAL_SCRIPTS.md#tutorial-2-file-operations)
- Practice file tasks

**Day 4: Code Generation**

- Read [Code Editing](USER_GUIDE.md#code-editing)
- Watch [Code Generation Tutorial](VIDEO_TUTORIAL_SCRIPTS.md#tutorial-3-code-generation)
- Generate some code

**Day 5: Autonomous Agents**

- Read [Autonomous Agents](USER_GUIDE.md#autonomous-agents)
- Watch [Agent Tutorial](VIDEO_TUTORIAL_SCRIPTS.md#tutorial-4-autonomous-agents)
- Try a simple agent task

**Day 6: Exploration**

- Browse [Features Guide](FEATURES.md)
- Try one new feature
- Experiment freely

**Day 7: Review & Practice**

- Review [FAQ](FAQ.md)
- Practice daily workflows
- Customize settings

### Path 2: Power User (2 weeks)

**Week 1: Foundations**

- Complete Beginner path
- Master keyboard shortcuts
- Customize your setup

**Week 2: Advanced Features**

- Day 8: [Workflow Automation](USER_GUIDE.md#workflow-automation)
- Day 9: [MCP Integration](USER_GUIDE.md#mcp-integration)
- Day 10: [Browser Automation](USER_GUIDE.md#browser-automation)
- Day 11: [Document Processing](USER_GUIDE.md#document-processing)
- Day 12: [Terminal Integration](USER_GUIDE.md#terminal-integration)
- Day 13: Advanced agent tasks
- Day 14: Build custom workflows

### Path 3: Developer (3 weeks)

**Weeks 1-2: Power User Path**

- Complete Power User path first

**Week 3: Development**

- Day 15: Read [Architecture](../ARCHITECTURE.md)
- Day 16: Review [API Documentation](../apps/web/API_DOCUMENTATION.md)
- Day 17: Set up development environment
- Day 18: Create custom MCP server
- Day 19: Build API integration
- Day 20: Contribute to project
- Day 21: Share with community
