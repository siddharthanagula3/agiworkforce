# AGI Workforce — Viral Video Scripts

**Platform: TikTok · Instagram Reels · YouTube Shorts**
**Research source: `docs/marketing/viral-content-research-report.md`**

All scripts follow the proven formula:

- **Hook [0-3s]** — One punchy line, no intro, no "hey guys"
- **Pain [3-10s]** — Why you need this RIGHT NOW
- **Demo [10-35s]** — Show it working
- **Amplify [35-50s]** — The part that blew my mind
- **CTA [50-57s]** — Single, simple action
- **Loop bait [57-60s]** — End on something re-watchable

# PLATFORM-SPECIFIC ADAPTATION GUIDE

## TikTok

- Lead with the most controversial or shocking hook
- Use trending sounds (change weekly — check TikTok Creative Center)
- Caption: Hook line first + hashtags (keep under 8)
- Best posting time: 7-9pm local, Tue-Thu
- 30-60 seconds optimal
- Reply to comments within 1 hour for algorithmic boost

## Instagram Reels

- Use text overlay on the first frame (60% watch without sound)
- Stories + DM CTA ("Send DM for the link")
- Caption: More conversational, can be longer, use line breaks
- Cross-post to Facebook Reels automatically
- Best posting time: 11am-1pm, Mon-Wed
- 15-30 seconds performs best for reach, 45-60 for saves

## YouTube Shorts

- Title IS the hook — write it like a clickbait headline
- First frame must have text (no black screens)
- End screen with subscribe prompt
- Longer scripts (60s) perform better on YouTube than TikTok
- SEO matters: include keywords in title + description
- Best for: developer-focused content (Script 4, 9)
