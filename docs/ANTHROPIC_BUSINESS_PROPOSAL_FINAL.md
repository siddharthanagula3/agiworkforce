### Capabilities Included

**Agent System:**

- Multi-agent orchestration with task queuing and dependency management
- Plan-act-observe execution loops with outcome tracking
- Learning system that improves from execution results
- RAG (Retrieval-Augmented Generation) for knowledge retrieval

**Automation:**

- Browser control via Chrome DevTools Protocol + Playwright
- Desktop automation (Windows UIA, macOS Accessibility APIs)
- Screen capture and vision-based action planning
- File system operations with permission scoping

**LLM Integration:**

- Multi-provider router (Claude, GPT, Gemini, Ollama, and others)

- Intelligent caching and cost management

**Infrastructure:**

- SQLite local database with encryption
- WebSocket real-time communication
- CI/CD with automated multi-platform builds
