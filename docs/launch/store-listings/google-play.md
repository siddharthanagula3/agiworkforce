## Full description (4,000 chars)

```
The AI app that doesn't lock you to one model.

Switch between Claude, GPT, Gemini, and 10+ providers mid-conversation. The same thread works across all of them. One UI. One conversation. Your choice.

🟢 KEY FEATURES

• Multi-provider chat — Anthropic Claude, OpenAI GPT, Google Gemini, xAI Grok, DeepSeek, Mistral, Qwen, Moonshot Kimi, Ollama (local), and more
• Bring your own key (BYOK) — paste your provider API key and pay them directly. No middleman markup.
• Local LLM support — point at your computer running Ollama for fully-offline AI
• Cross-device sync — start a conversation on Android, finish it on desktop
• Voice transcription — Whisper-grade speech-to-text built in
• Computer use (premium tier) — let the agent navigate web pages, fill forms, summarize screens

🟢 WHY IT'S DIFFERENT

Most AI apps ship with one provider's models locked in. We built the cross-vendor plumbing — payload normalization, schema cleanup, reasoning-effort routing — so any model is one tap away. Anthropic doesn't accept user keys. OpenAI doesn't ship a Gemini option. We do all three, plus the long tail.

🟢 PRICING

• Local-only — Free forever. Run Ollama on your laptop. No account needed.
• BYOK — Free forever. Bring your own API keys. We never see them.
• Hobby — $5/mo. Managed cloud, limited credits, the small models we route to.
• Pro / Max — Coming after security audit. Join the waitlist.
• Enterprise — SSO, SCIM, custom retention. Contact sales.

🟢 PRIVACY

• We never send your messages to a model you didn't pick.
• BYOK keys are stored locally (Android Keystore) and used only for the API call you triggered.
• No training on your conversations. Ever.

🟢 SUPPORT

• Docs: https://agiworkforce.com/docs
• Email: support@agiworkforce.com
```

## Privacy & Data Safety

Required Play disclosures:

- **Personal info**: email + name (account creation only)
- **App activity**: in-app messages — collected to deliver to LLM provider on user's behalf, not shared with third parties beyond the provider the user picked
- **Device IDs**: not collected
- **Encryption**: data encrypted in transit (TLS) and at rest (Supabase row-level)
- **Data deletion**: in-app at Settings → Account → Delete
