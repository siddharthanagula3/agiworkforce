## Description (4,000 chars)

```
AGI Workforce is the AI assistant that doesn't lock you to one model.

Switch between Claude, GPT, Gemini, and 10+ providers mid-conversation. The same chat thread works across all of them — Claude for nuance, GPT for tools, Gemini for vision, Llama for offline. One UI. One conversation. Your choice.

KEY FEATURES

• Multi-provider chat — Anthropic Claude, OpenAI GPT, Google Gemini, xAI Grok, DeepSeek, Mistral, Qwen, Moonshot Kimi, Ollama (local), and more
• Bring your own key — paste your Anthropic / OpenAI / Google API key and pay your provider directly. No marketplace markup.
• Local LLM support — point at your laptop running Ollama or LMStudio for fully-private conversations
• Cross-device sync — start a conversation on iPhone, finish it on the desktop app
• Voice transcription — speak prompts via Whisper or your platform's built-in voice
• Computer use (premium tier) — let the agent navigate web pages, fill forms, summarize screens

WHY IT'S DIFFERENT

Most AI apps ship with one provider's models locked in. We've built the cross-vendor plumbing — payload normalization, schema cleanup, reasoning-effort routing — so any model is one tap away. Anthropic doesn't accept user keys. OpenAI doesn't ship a Gemini option. We do all three, plus the long tail.

PRICING

• Local-only — Free forever. Run Ollama on your laptop. No account needed.
• BYOK — Free forever. Bring your own API keys. We never see them.
• Hobby — $5/mo. Managed cloud, limited credits, the small models we route to.
• Pro / Max — Coming after security audit. Join the waitlist.
• Enterprise — SSO, SCIM, custom retention. Contact sales.

PRIVACY

• We never send your messages to a model you didn't pick.
• BYOK keys are stored locally (Keychain) and used only for the API call you triggered.
• No training on your conversations. Ever.
• Optional cross-device sync uses your Supabase account; your data stays in your row.

SUPPORT

• Visit https://agiworkforce.com for docs and support
• Email support@agiworkforce.com
```
