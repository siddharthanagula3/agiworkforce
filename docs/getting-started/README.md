## Prerequisites

- Node.js 22.12.0+
- pnpm 9.15.3+
- Rust (for desktop development)
- Git

## Quick Start

```bash
# Clone the repository
git clone https://github.com/siddhartha/agiworkforce.git
cd agiworkforce

# Install dependencies
pnpm install

# Start desktop development
pnpm dev:desktop
```
