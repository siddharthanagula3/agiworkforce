## Production Notes

### Equipment Needed

- Screen recording software (OBS, ScreenFlow, Camtasia)
- High-quality microphone
- Video editing software
- AGI Workforce installed and configured
- Demo accounts and sample data

### Video Settings

- **Resolution**: 1920x1080 (1080p minimum)
- **Frame rate**: 30 or 60 fps
- **Codec**: H.264
- **Bitrate**: 8-12 Mbps
- **Format**: MP4

### Recording Tips

1. Close unnecessary applications
2. Disable notifications
3. Use a clean browser profile
4. Prepare sample files and folders beforehand
5. Test audio levels before recording
6. Record in a quiet environment
7. Speak clearly and at moderate pace
8. Pause between major sections for easier editing

### Editing Guidelines

1. Add intro animation (0-5 seconds)
2. Include chapter markers
3. Add callouts and highlights for key UI elements
4. Use zoom/pan to focus attention
5. Add captions/subtitles
6. Include end card with next videos (last 10 seconds)
7. Background music (subtle, non-distracting)
8. Export with YouTube-optimal settings

### Publishing Checklist

- [ ] Title includes "AGI Workforce" and main topic
- [ ] Thumbnail shows clear benefit/outcome
- [ ] Description includes timestamps
- [ ] Tags include relevant keywords
- [ ] Add to appropriate playlist
- [ ] Enable captions
- [ ] Add cards linking to related videos
- [ ] Pin comment with links to documentation
- [ ] Share on social media
