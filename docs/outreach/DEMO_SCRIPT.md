# AGI Workforce — Demo Script (2:30)

**Goal**: Show a decision-maker that this is a real, working product — not a prototype. Lead with outcomes, not features.

---

## Setup Before Recording

- Desktop app running with clean state
- At least 2 LLM providers connected (e.g., OpenAI GPT-4 + Claude + a local Ollama model)
- Mobile companion paired (or simulator showing the app)
- Chrome extension installed
- Have a real task ready (e.g., "draft a contract review" or "analyze this quarterly report")

---

## Scene 1: The Hook (0:00 - 0:15)

**[Screen: Desktop app, clean chat interface]**

> "This is AGI Workforce. One native desktop app. Every major AI model. Full desktop control. And 150 skills that have nothing to do with code."
>
> "Let me show you what that actually means."

---

## Scene 2: Multi-Model in Action (0:15 - 0:45)

**[Screen: Model selector dropdown showing 24 providers]**

> "I'm connected to OpenAI, Anthropic, Google, Mistral, and Llama running locally through Ollama. No subscriptions stacking up — I bring my own API keys."

**[Action: Switch from GPT-4 to Claude mid-conversation]**

> "I can switch models mid-conversation. Same context, different brain. Try doing that in ChatGPT."

**[Action: Show Ollama running locally]**

> "And this model is running entirely on my machine. Nothing leaves my laptop."

---

## Scene 3: Non-Coding AI Skills (0:45 - 1:15)

**[Screen: Skills panel showing categories — Legal, Medical, Finance, Education]**

> "150 built-in skills. Not coding tools — real professional workflows."

**[Action: Use a legal skill — e.g., "Review this NDA for red flags"]**

> "I just asked it to review a contract. It flagged three non-standard clauses and suggested alternatives. A lawyer built this skill. It took 10 seconds."

**[Action: Quick flash through other skills — medical differential, financial modeling, lesson planning]**

> "Medical differentials. Financial projections. Lesson plans. Every profession, not just developers."

---

## Scene 4: Desktop Automation (1:15 - 1:40)

**[Screen: Agent running a multi-step desktop task]**

> "Now the part no one else has. AGI Workforce controls your actual desktop."

**[Action: Agent opens a browser, navigates to a site, extracts data, pastes into a spreadsheet]**

> "That wasn't a browser extension. That wasn't an API call. The agent is using your computer like you would — screen, keyboard, mouse. Natively."

## Scene 5: Mobile Agent Oversight (1:40 - 2:05)

**[Screen: Split — desktop on left, phone on right]**

> "But here's the real question with autonomous agents: do you trust them?"

> "No competitor has this."

**[End card: AGI Workforce logo + URL + "Request a demo" CTA]**

---

## Recording Tips

- **Resolution**: 1920x1080 minimum, 4K preferred
- **Pacing**: Don't rush. Let each action land before narrating the next
- **Cursor**: Use a large, visible cursor. Highlight clicks
- **Audio**: Record voiceover separately for clean audio. No typing sounds
- **Length**: Aim for exactly 2:00-2:30. Cut ruthlessly. Every second must earn its place
- **Music**: Subtle, minimal background track. Nothing distracting
- **Tools**: OBS for screen recording, DaVinci Resolve or ScreenFlow for editing
