## Tips for Productivity

### 1. Learn the Command Palette (`Cmd/Ctrl + K`)

This is the most important shortcut! It provides fuzzy search access to:

- All commands
- Recent files
- Conversations
- Settings
- Workflows
- Everything!

**Pro Tip**: Type `/` in command palette to see categories.

### 2. Use Vim-Style Navigation (J/K)

Navigate messages without leaving the keyboard:

- `J` or `↓` - Next message
- `K` or `↑` - Previous message
- `G G` - First message
- `Shift + G` - Last message

Enable in: Settings → Keyboard → Vim Mode

### 3. Multi-Cursor Editing

Edit multiple locations simultaneously:

1. `Cmd/Ctrl + D` - Select next occurrence
2. `Cmd/Ctrl + U` - Undo last cursor
3. `Alt + Click` - Add cursor at click
4. `Cmd/Ctrl + Alt + ↑/↓` - Add cursor above/below

### 4. Quick Model Switching

Don't interrupt your flow:

1. `Alt + P` - Open model selector
2. Type first letters of model
3. `Enter` - Select

Example: `Alt + P`, `g4`, `Enter` selects GPT-4

### 5. Chain Shortcuts

Some shortcuts can be chained:

- `Cmd/Ctrl + K`, `Cmd/Ctrl + F` - Format selection
- `Cmd/Ctrl + K`, `Cmd/Ctrl + C` - Add line comment
- `Cmd/Ctrl + K`, `Cmd/Ctrl + U` - Remove line comment

### 6. Context-Aware Shortcuts

Same shortcut, different action based on context:

- `Cmd/Ctrl + F` in chat: Search messages
- `Cmd/Ctrl + F` in editor: Find in file
- `Cmd/Ctrl + F` in browser: Find in page

### 7. Create Your Own Workflows

Assign shortcuts to custom workflows:

1. Create workflow in Automation
2. Edit workflow settings
3. Assign keyboard shortcut
4. Use `Cmd/Ctrl + Shift + <key>` to trigger

## Accessibility

### Screen Reader Support

Navigate with keyboard when using screen readers:

- `Tab` - Next element
- `Shift + Tab` - Previous element
- `Enter` - Activate
- `Space` - Toggle/Select
- `Arrow keys` - Navigate lists/menus

### Keyboard-Only Mode

Enable complete keyboard navigation:

1. Settings → Accessibility
2. Enable "Full Keyboard Navigation"
3. Use `Tab` to navigate all elements

### Custom Key Bindings

For accessibility needs, all shortcuts can be:

- Remapped to single keys
- Assigned to different modifiers
- Disabled entirely
- Exported/imported for consistency

## Quick Practice Exercise

Try this 2-minute exercise to learn key shortcuts:

1. Open command palette: `Cmd/Ctrl + K`
2. Type "new conversation" and press Enter
3. Send a message: Type "Hello", then `Cmd/Ctrl + Enter`
4. Switch models: `Alt + P`, select a model
5. Search: `Cmd/Ctrl + F`, search for "Hello"
6. Navigate: Press `K` to go to previous message
7. Close search: `Escape`

**Congratulations!** You've learned 7 essential shortcuts.
