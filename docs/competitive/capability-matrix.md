# AGI Workforce — Canonical Capability Matrix

_Created: March 19, 2026 — Last updated: March 19, 2026 (Q2 execution close)_

This document is the canonical status baseline for AGI Workforce capabilities.

It is intentionally conservative.

## Status Definitions

- **SHIPPED** — capability is clearly implemented in the repo and exposed through real product code or routes.
- **PARTIAL** — capability exists, but has known scope, reliability, integration, or polish gaps.
- **IN_PROGRESS** — capability has active implementation footing in the repo and is explicitly on the active execution path.
- **BLOCKED** — capability is currently stopped by a named dependency or unresolved prerequisite.
