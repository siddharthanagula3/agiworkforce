# @agiworkforce/types

Shared TypeScript types used across all surfaces.

## Usage

```typescript
import type { ChatMessage, Provider } from '@agiworkforce/types';
```

## Contents

- `model-catalog.ts` — Provider, ModelMetadata, ModelCapabilities
- `chat.ts` — ChatMessage, Conversation, ConversationSummary
- `audit.ts` — AuditEvent, AuditSurface, AuditAction
- `cross-device.ts` — CrossDeviceThread, DevicePairing
- `signaling.ts` — SignalingEvent, PairingCode
- `conversation.ts` — MessageRole, ArtifactType, ApprovalRequest
