================================================================================
                         SUPPORT & DEBUGGING
================================================================================

CAN'T RUN TESTS?
  1. Check environment variables (.env.local)
  2. Verify test database is accessible
  3. Run: pnpm exec playwright test --list
  4. Check app is running: npm run dev

TEST FAILURES?
  1. View screenshot: playwright-report/
  2. View HTML report: pnpm exec playwright show-report
  3. Run with debug: PWDEBUG=1 pnpm test:e2e
  4. Check logs in stdout

NEED HELP?
  1. Read QUICK_START.md (1 minute)
  2. Read CRITICAL_FLOWS_GUIDE.md (5 minutes)
  3. Read README.md (10 minutes)
  4. Enable debug mode and inspect

STILL STUCK?
  - Check Playwright docs: https://playwright.dev
  - Review page objects for selector changes
  - Check if app features have changed
  - Run with --headed to see browser
