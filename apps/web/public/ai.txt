# AGI Workforce - AI & Automation Platform
# Website: https://agiworkforce.com

## Overview
AGI Workforce is a comprehensive platform for deploying autonomous AI agents capable of:
- Automated task execution and workflow automation
- Computer use and screen interaction
- Multi-step reasoning and tool usage
- Integration with 9+ LLM providers
- Desktop and web-based interfaces

## Key Features
- **Agentic Chat**: Conversational AI with tool-calling capabilities
- **Computer Use**: Screen capture, OCR, keyboard/mouse input automation
- **Multi-LLM Support**: Claude, GPT-4, Gemini, Llama, and more
- **Workflow Automation**: Create complex automation workflows
- **Local-First**: API keys stored securely in OS keyring
- **Open Ecosystem**: Browser extension, desktop app, web platform
