# @openai/agiworkforce-responses-api-proxy

<p align="center"><code>npm i -g @openai/agiworkforce-responses-api-proxy</code> to install <code>agiworkforce-responses-api-proxy</code></p>

This package distributes the prebuilt [Agiworkforce Responses API proxy binary](https://github.com/openai/codex/tree/main/agiworkforce-rs/responses-api-proxy) for macOS, Linux, and Windows.

To see available options, run:

```
node ./bin/agiworkforce-responses-api-proxy.js --help
```

Refer to [`agiworkforce-rs/responses-api-proxy/README.md`](https://github.com/openai/codex/blob/main/agiworkforce-rs/responses-api-proxy/README.md) for detailed documentation.
