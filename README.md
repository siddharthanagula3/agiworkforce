# AGI Workforce (Developer Preview)

Tauri + React desktop agent workspace with autonomous execution, tool approval rails, and a full productivity ecosystem.

## 🚀 Key Feature Pillars (Developer Preview)

### 1. Workspace & Intelligence (Core)

- **Unified Agentic Chat**: Immersive chat interface with real-time streaming, status trails, and reasoning accordions.
- **Artifacts & Visualization**: Rich rendering for code blocks, diagrams, and multimedia generated by agents.
- **Terminal Workspace**: Multi-session terminal emulator with AI-powered command suggestions and error explanations.
- **Deep Git Integration**: Built-in `git2` repository management (clone, push, pull, status, etc.) via the terminal and agent tools.
- **Workspace Intelligence**: Symbol indexing, reference finding, and dependency analysis for large codebases.
- **Media Lab**: High-fidelity UI for managing and previewing agent-generated media.

### 2. Autonomous Capabilities (Core)

- **Agent Orchestration**: Autonomous system for planning and executing multi-step tasks across the filesystem and web.
- **Tool Approval Rails**: Granular "Safe Mode" prompts for user consent on high-risk operations (terminal, file writes, git).
- **Computer Use**: Vision-guided system-wide UI automation and browser control.
- **OCR & Vision Integration**: Real-time screen processing and text extraction for automating native applications.

### 3. Productivity Ecosystem (Agent-Only Tools)

> [!NOTE]
> These features are currently available to the agent via tool-calling. Dedicated UI suites are planned for future releases.

- **Email Suite**: IMAP/SMTP integration for automated mailbox management and contact synchronization.
- **Calendar Management**: OAuth2 support for Google/Outlook calendars with event orchestration.
- **Cloud Storage**: Unified interface for Google Drive, Dropbox, and OneDrive operations.
- **Enterprise Messaging**: Connected clients for Slack, Microsoft Teams, and WhatsApp Business API.

### 4. Experimental & Future Scope

> [!TIP]
> These modules are under active development and located in `future_scope/`.

- **AI Employee Library**: Specialized pre-built agents for recurring roles (e.g., Code Reviewer, Content Strategist).
- **Workforce Marketplace**: A viral system for sharing and discovering autonomous workflows.
- **ROI & Cost Analytics**: Real-time token usage tracking and automation efficiency dashboards.

## 🛠 Prerequisites

- **Node.js**: ≥20.11.0 (pnpm 9.x)
- **Rust**: 1.90.0 (pinned in `rust-toolchain.toml`)
- **Platform Dependencies**:
  - **macOS**: Xcode Command Line Tools
  - **Windows**: Visual Studio Build Tools (C++)
  - **Linux**: WebKit2GTK and build-essential

## 🚦 Getting Started

1. **Install Dependencies**:

   ```bash
   pnpm install
   ```

2. **Configure Environment**:
   Copy `apps/desktop/.env.example` to `apps/desktop/.env` and set your API keys.

3. **Run Development Mode**:
   ```bash
   pnpm dev:desktop
   ```

## 🏗 Build & Test

```bash
pnpm build:desktop      # Production bundle
pnpm test                # Run Vitest + Cargo tests
pnpm lint                # Enforce code style
```

## 📂 Repository Layout

- `apps/desktop`: Core Tauri application (React frontend + Rust backend)
- `packages/types`: Shared TypeScript definitions
- `packages/utils`: Cross-workspace utility helpers
- `services/*`: Experimental microservices (api-gateway, signaling)
